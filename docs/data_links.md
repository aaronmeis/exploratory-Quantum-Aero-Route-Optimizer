# Data Sources: Quantum-Aero-Route-Optimizer

## Recommended Dataset
**Kaggle: Airlines, Airport, and Flight Routes**
- Link: [Global Air Transportation Network](https://www.kaggle.com/open-flights/airports-train-stations-and-ferry-terminals)
- Description: Comprehensive data on airports, airlines, and routes. Ideal for building large-scale network graphs for optimization.

## Instructions
1. Download the airline route dataset.
2. Use the IATA/ICAO codes and coordinates to build distance-weighted graphs in `NetworkX`.
3. Place raw data in `data/`.
